# OSSPrac 1조 Sparkle팀 소개 페이지

## 프로젝트 개요
Flask, Docker, HTML, CSS등을 활용하여 오픈소스 소프트웨어 실습 수업의 팀원을 소개하는 웹 페이지입니다.

## 기술 스택
- **Frontend**: HTML, CSS, Javascript
- **ect**: flask, docker, docker-compose

## 사용 흐름

1. **메인 페이지** (`/index`)
   - 메인 팀 소개 화면
   - 팀원 개인정보 및 팀 소개 영상 첨부

2. **팀원 추가 페이지** (`/input`)
   - 개인정보 입력 화면
   - 각종 유효성검사 정규식 적용

3. **연락처 페이지** (`/contact`)
   - 연락처 정보 기재
   - 문의내용 입력 폼 구현
